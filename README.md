# 🛍️ Exum Store

Exum es una tienda e-commerce multinicho desarrollada para la venta de productos a nivel nacional con proyección de escalamiento internacional.  
El objetivo del proyecto es construir una plataforma moderna, escalable y optimizada para integrar múltiples categorías de productos, automatización de pedidos y futuros sistemas de dropshipping mediante APIs como Dropi y servicios similares.

## 🚀 Visión del proyecto

La plataforma está diseñada para evolucionar progresivamente desde una tienda nacional hacia un ecosistema de comercio electrónico con integración de proveedores externos, automatización logística y expansión internacional.

Exum busca combinar:
- Rendimiento
- Escalabilidad
- Experiencia visual moderna
- Arquitectura flexible
- Integraciones futuras con servicios de fulfillment y dropshipping
